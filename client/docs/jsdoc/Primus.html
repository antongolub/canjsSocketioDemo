Class: Primus

Primus

new Primus(url, options)

Primus in a real-time library agnostic framework for establishing real-time connections with servers. Options: - reconnect, configuration for the reconnect process. - manual, don't automatically call `.open` to start the connection. - websockets, force the use of WebSockets, even when you should avoid them. - timeout, connect timeout, server didn't respond in a timely manner. - ping, The heartbeat interval for sending a ping packet to the server. - pong, The heartbeat timeout for receiving a response to the ping. - network, Use network events as leading method for network connection drops. - strategy, Reconnection strategies. - transport, Transport options. - url, uri, The URL to use connect with the server.
Parameters:
Name Type Description
url String The URL of your server.
options Object The configuration.
